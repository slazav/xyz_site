=== Объекты

Объекты могут создаваться и правиться разными пользователями. Старые
версии хранятся в архивной базе, каждый исправленный объект имеет ссылку
на предыдущую версию. Удаленные объекты сохраняются в основной базе с
полем del=1.

== Информация об объекте в базе данных

*   _id - идентификатор объекта
* cuser - пользователь, создавший объект
* ctime - время создания
* muser - пользователь, вносивший последнее исправление
* mtime - время последнего исправления
* duser - пользователь, удаливший/восстановивший объект
* dtime - время последнего удаления/восстановления
* prev  - id предыдущей версии в архиве
* del   - поле для маркировки удаленных объектов

Другие поля, в зависимости от типа объекта.

Для хранения счетчиков используется коллекция counters с полями:
*   _id:  название базы объектов
*   seq:  счетчик

== Типы объектов

* новость (коллекции news и news.arc)

  * текст
  * теги

* поход
  * заголовок
  * текст
  * теги
  * участники
  * реки

* геообъект
  * заголовок
  * текст
  * теги
  * GeoJSON
  * походы

* материал

  * ссылка
  * автор
  * тип
